Base de Conhecimento de Linguagens de Programação

Explore uma coleção completa com linguagens modernas, históricas, funcionais, para sistemas, mobile e até para microcontroladores como Arduino e ESP32. Cada item inclui uma breve descrição, nível de dificuldade, ano de criação e links de documentação.

Utilize a busca para filtrar linguagens por nome, termos técnicos ou tags como backend, mobile, IA, funcional, embarcados e outras áreas.
